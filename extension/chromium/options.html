Настройки Sellico Live

Расширение собирает только реальные сигналы из вашего кабинета Wildberries и передает их в Sellico.

Подключение через Sellico Введите данные Sellico один раз. Пароль не сохраняется; расширение сохранит только access token в своем защищенном storage.
